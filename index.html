Rhino & AutoCAD Best Practices

🦏Rhino

Reducing Save Times

The following settings are intended to reduce the time it takes Rhino to save files to the network.

  1. Go to the Tools pulldown and select Options
    • Alternatively, enter _Option in Rhino’s command line.
  1. In the left sidebar of the next window select Rhino Options → Advanced
  1. In the top search bar where it says “Type to filter keys…” input:
    1. Rhino.Options.FileSettings.WriteLocalTempFileWhenSaving ∟ Set the Value → True
    1. Rhino.Options.Advanced.UseCompressionWhenSaving

      ∟ Set the Value → False

📏AutoCAD

Using the LOCAL AutoCAD profile

If you are using AutoCAD you should be using the LOCAL AutoCAD profile. That way AutoCAD is not reaching out over your internet connection for things like fonts, hatch patterns, custom commands, tool palettes, etc. The local profile uses a local copy of all that stuff which is way faster.

If you're not already using it, here's how to set it up (don't panic it's easy):

  1. Make sure AutoCAD is not running
  1. Make sure the VPN is active and you can see the Z drive
  1. Browse to "Z:\CAD-Common\AutoCAD 2020"
  1. Double-click the file called "Update SNO LOCAL AutoCAD Folders.bat"
  1. A whole mess of gibberish will scroll by and then the box will close.
  1. Launch AutoCAD from the new icon on your desktop called "SNO LOCAL AutoCAD 2020"

Speeding up the Sheet Set Manager

Everyone on the project should do this. If one person doesn't use the following settings, no one benefits.

You may find that these settings have been set automatically; It’s still worth having a look!

Change these settings in AutoCAD to cut down on the amount of SSM generated traffic:

  1. Type SSMSHEETSTATUS and press ENTER, set it to 0. (this one does the most good)
    • Sheet lock status will no longer be displayed in the sheet set manager. The files will still be locked of course, but you won't know until you try to open it (just like in the good old days).
  1. Type SSMAUTOOPEN and press ENTER, set it to 0.
  1. Type SSLOCATE and press ENTER, set it to 0.
    • Steps 2 and three work together (both 0 or both 1). When set to 0, opening a file that is a member of a sheet set will no longer also open the sheet set automatically. So you can work on a file without having the sheet set open at all. This is great if you just need to draft or edit text, etc. However, If you need to rename sheets, place views, place tags etc. you'll need to have the sheet set open of course - which you can still do manually.

When you're not going to be working in a sheet set for a while or when you're done for the night, always exit the sheet set by right clicking on the sheet set name in the SSM and selecting "Close Sheet Set". That's one less person sending and receiving sheet set data.

When doing things that tax the sheet set (publishing in particular, but also renaming sheets, archiving...), try to organize that with the team as best you can.  Have one person do the publishing or renaming and have everyone else exit the sheet set... if that's possible.

Managing XREFs

All XREF paths in AutoCAD should start with a mapped drive letter, for example P:\ or O:\. This is a form of “relative referencing”. The alternative to relative referencing is “explicit referencing.” For example: “\\anf-d236\..\example.png”.When we use explicit references to a specific server location, we allow AutoCad to ignore the infrastructure that is in place to retrieve information efficiently.

General

  • Connect your computer to your router at home with an ethernet cable, don't rely on wireless. Wireless is slower all around and is subject to dropouts and random VPN issues.
  • When connected to the internet via ethernet make sure you disable WIFI. The ambiguity of communicating via two network connections is a common cause of VPN problems.
  • Get the fastest synchronous internet service available to you that you can afford. (synchronous meaning the same speed for upload and download). Verizon Fios fiber is a great option (close to 1 Gigabit up and down for +/- $70 a month). Ask your current provider about synchronous service. It might also be worth getting a separate line for your office if you share your internet with a whole household.
  • If possible, don't stream music or video while you're working (spotify, itunes, netflix, Roku, Apple TV, etc will all eat as much bandwidth as they can - stealing it away from you).
  • Disable WiFi calling on your cell phone if you use it a lot during the workday, home routers generally prioritize voice traffic and so if you're on the phone, your internet slows down.
